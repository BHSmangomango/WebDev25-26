Web Technologies

HTML

HTML stands for Hypertext Markup Language that creates web pages. It consists of Headings, titles, paragraphs, images, links, etc. Its make a visual presentation of a web page for users.

CSS

CSS stands for Cascading Style Sheets in which is a stylesheet language for your web page. It shows colors, background, layout, etc. It enhances user experience and aesthetics for the website.

JavaScript

JavaScript is a dynamic programming language used primarily to create interactive and dynamic web content. For example, it controls everything from animated graphics and photo slideshows to interactive forms and autocomplete suggestions on a website. As one of the core technologies of the World Wide Web, JavaScript works with HTML and CSS to create the modern, dynamic user experiences we expect today.

Bootstrap

the process of booting up a computer by using a small, initial program to load a larger one
